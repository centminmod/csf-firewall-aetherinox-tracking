# CSF Firewall Fork Tracking

This repository tracks and documents all changes between the official [CSF Firewall v15.00](https://github.com/centminmod/configserver-scripts/tree/main/csf) (GPLv3) and the [Aetherinox fork](https://github.com/Aetherinox/csf-firewall).

## Main Documentation

### [aetherinox-fork-changes.md](aetherinox-fork-changes.md)

Comprehensive change documentation including:

| Section | Description |
|---------|-------------|
| **Summary** | File count statistics (modified, new, removed) |
| **Repository State** | Current HEAD commits for both repositories |
| **Change Statistics** | Lines added/removed, file size differences |
| **New Files** | Files unique to the fork (branding, JS, shell scripts) |
| **Modified Files** | Detailed commit history for changed files |
| **Feature Summary** | New CLI commands, UI enhancements, security features |
| **Detailed Change Analysis** | Code-level analysis of key commits |
| **Configuration Options** | All new config settings with defaults |
| **Commit Reference** | Quick lookup of important commits |

All commit hashes are hyperlinked to their GitHub pages for easy inspection.
